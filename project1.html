S-H2O: Smart Hydroponics Irrigation App

Project Description

S-H2O is a smart irrigation application designed for hydroponic lettuce farming. It allows remote monitoring of water and nutrient levels, automates water-saving irrigation, and optimizes decision-making using a threshold algorithm combined with linear regression.

Key Features

  • Remote monitoring of water and nutrient levels
  • Automated water-saving irrigation system
  • Decision-making via threshold algorithm & linear regression
  • User-friendly mobile interface for real-time control

Technologies Used

Flutter, Firebase, Arduino ESP32

Future Improvements

  • Real-time alerts for nutrient or water shortage
  • AI-driven predictive irrigation scheduling
  • Integration with additional sensors for temperature & humidity
