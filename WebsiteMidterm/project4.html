Project Description

This future project aims to create an AI-powered running analytics system that collects data from runners (distance, pace, heart rate, cadence) and provides personalized training plans, injury risk alerts, and performance predictions.

Planned Features

  • Real-time pace, distance, and heart rate monitoring
  • AI-generated personalized weekly training plans
  • Injury risk prediction based on historical and live data
  • Progress visualization dashboards
  • Option to simulate performance improvement over time

Planned Technologies

  • Python (ML models & data analysis)
  • Flutter or Web Dashboard
  • Wearable sensors / Arduino / ESP32 for data collection
  • Firebase or Cloud Database
