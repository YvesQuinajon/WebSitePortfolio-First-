Hangman Game (Java)

Project Overview

The Hangman Game is a Java-based console application where the player guesses letters to uncover a hidden word. The game tracks wrong guesses, updates the hangman figure after each mistake, and ends when the player wins or exhausts all attempts.

Key Features

  • Random word selection
  • Input validation and correct/incorrect letter checking
  • ASCII hangman figure updates after each wrong attempt
  • Win or lose game states
  • Replay option after finishing a round

Technologies Used

Java, Loops, Conditionals, Arrays, OOP Basics
